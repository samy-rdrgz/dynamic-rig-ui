# Dynamic Rig-ui

Création dynamique d'interface utilisateur pour RIG Blender. Se base sur les bones collections, affiche leur visibilité, les customs properties, les mask_modifiers des objets enfants, et différents boutons de switch IK FK.

## Installation

1. Download ZIP
2. Blender > Edit > Preferences > Add-ons > Install
3. Activer l'addon

## Compatibilité

- Blender 5.0+
